# astal-bluetooth

Fedora RPM packaging for Astal's Bluetooth integration library.

Current packaging target:

- pinned `Aylur/astal` snapshot commit `41b5029` (`snapshot_date 20260319`)
- provides the `AstalBluetooth-0.1` GIR/typelib for BlueZ-backed widgets
- intended as an optional AGS/Astal integration for Bluetooth status and device views
