# astal-wireplumber

Fedora RPM packaging for Astal's WirePlumber integration library.

Current packaging target:

- pinned `Aylur/astal` snapshot commit `41b5029` (`snapshot_date 20260319`)
- provides the `AstalWp-0.1` GIR/typelib and `libastal-wireplumber`
- intended as an optional AGS/Astal integration for PipeWire and audio widgets
