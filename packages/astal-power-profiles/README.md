# astal-power-profiles

Fedora RPM packaging for Astal's power-profiles library and CLI bindings.

Current packaging target:

- pinned `Aylur/astal` snapshot commit `67ddc83` (`snapshot_date 20260430`)
- provides the `AstalPowerProfiles-0.1` GIR/typelib plus the `astal-power-profiles` CLI
- intended as an optional AGS/Astal integration for power profile widgets
