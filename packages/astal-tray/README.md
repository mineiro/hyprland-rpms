# astal-tray

Fedora RPM packaging for Astal's status-notifier tray bindings.

Current packaging target:

- pinned `Aylur/astal` snapshot commit `67ddc83` (`snapshot_date 20260430`)
- provides the `AstalTray-0.1` GIR/typelib plus the `astal-tray` CLI
- depends on the packaged `appmenu-glib-translator` library
