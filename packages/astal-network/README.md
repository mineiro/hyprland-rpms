# astal-network

Fedora RPM packaging for Astal's NetworkManager integration library.

Current packaging target:

- pinned `Aylur/astal` snapshot commit `67ddc83` (`snapshot_date 20260430`)
- provides the `AstalNetwork-0.1` GIR/typelib and shared library
- intended as an optional AGS/Astal integration for wifi and wired widgets
