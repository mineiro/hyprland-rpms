# astal-battery

Fedora RPM packaging for Astal's battery library and CLI bindings.

Current packaging target:

- pinned `Aylur/astal` snapshot commit `41b5029` (`snapshot_date 20260319`)
- provides the `AstalBattery-0.1` GIR/typelib plus the `astal-battery` CLI
- intended as an optional AGS/Astal integration for UPower battery widgets
