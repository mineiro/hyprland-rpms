# astal-apps

Fedora RPM packaging for Astal's applications library and CLI bindings.

Current packaging target:

- pinned `Aylur/astal` snapshot commit `41b5029` (`snapshot_date 20260319`)
- provides the `AstalApps-0.1` GIR/typelib plus the `astal-apps` CLI
- intended as an optional AGS/Astal integration for app launchers and app lists
