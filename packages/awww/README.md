# awww

Packaging for `awww` (the Codeberg rename/successor of `swww`), an animated
Wayland wallpaper daemon and CLI.

Upstream:

- Git: `https://codeberg.org/LGFae/awww.git`
- Releases: `https://codeberg.org/LGFae/awww/releases`

Current packaging target:

- latest upstream release `v0.12.0`
- tagged release now builds the renamed `awww` / `awww-daemon` binaries directly
