# astal-auth

Fedora RPM packaging for Astal's authentication library and CLI bindings.

Current packaging target:

- pinned `Aylur/astal` snapshot commit `41b5029` (`snapshot_date 20260319`)
- provides the `AstalAuth-0.1` GIR/typelib plus the `astal-auth` CLI
- installs the upstream `astal-auth` PAM service file used by display-manager integrations
