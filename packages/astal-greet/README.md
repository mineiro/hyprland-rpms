# astal-greet

Fedora RPM packaging for Astal's greetd client library and CLI bindings.

Current packaging target:

- pinned `Aylur/astal` snapshot commit `67ddc83` (`snapshot_date 20260430`)
- provides the `AstalGreet-0.1` GIR/typelib plus the `astal-greet` CLI
- intended as an optional AGS/Astal integration for login and greeter flows
