# astal-notifd

Fedora RPM packaging for Astal's notification daemon library and CLI bindings.

Current packaging target:

- pinned `Aylur/astal` snapshot commit `67ddc83` (`snapshot_date 20260430`)
- provides the `AstalNotifd-0.1` GIR/typelib plus the `astal-notifd` CLI
- ships the upstream `io.astal.notifd` GSettings schema for notification state
