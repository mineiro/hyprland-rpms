# astal-mpris

Fedora RPM packaging for Astal's MPRIS library and CLI bindings.

Current packaging target:

- pinned `Aylur/astal` snapshot commit `41b5029` (`snapshot_date 20260319`)
- provides the `AstalMpris-0.1` GIR/typelib plus the `astal-mpris` CLI
- intended as an optional AGS/Astal integration for media player widgets
